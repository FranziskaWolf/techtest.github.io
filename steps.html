GTM

After watching tones of Youtube tutorials I have been able to set up the GTM on that site. Using the Tag Assistant Legacy for Chrome I have been able to check that it works. First step has been done. So now we can come to the tricky part. Using GTM I am able to create different variables, tags and trigger to add values coming from the webiste to data layer. What I think is supposed to happen: For example someone is buying something on that page and on the "thank you" site the website is submitting values like Order IDs, prices, anything like that. The plan now is to get thise values and keys into our data layer. This can work through setting up the variables in GTM ("variable type - URL - "order id").
