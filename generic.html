GTM and data layer

After watching tones of Youtube tutorials and documentations from Google I have been able to set up the GTM on that site. Using the Tag Assistant Legacy for Chrome I have been able to check that it works. First step has been done. So now we can come to the tricky part. Using GTM I am able to create different variables, tags and trigger to add values coming from the webiste to data layer. So the following variables need to be set up: eventAction, eventCategory, eventLabel, eventValue, transactionTotal. After that I needed to set up the Tag and trigger. Final step is creating an HTML-tag for the Data Layer Push.

Challenges

The problem now is that I can't really submit any "real" information. So I tried to show you what I supposed to happen. Therefor I chose an example using Books. Every part of the table is a key: bookAuthor, bookTitle and bookPrice. Every key has different values like "war and peace" or "pride and prejudice". I sat up those variables like mentioned above in my Google Tag Manager. Afterwards I used a script including "dataLayer.push()" to transfer the given information to the data layer.
